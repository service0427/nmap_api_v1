# Workspace Operational Rules

## Temporary & Test Files Policy
- **NO Scratch Files in Root**: NEVER write scratch scripts, temporary files, or test scripts directly to the project root directory.
- **Gitignored Locations Only**: Always place temporary files inside gitignored directories. For this repository, use the `temp/` directory (e.g. `/home/tech/aws/nmap_api_v1/temp/`).
- **Cleanup**: Delete temporary files in `temp/` immediately after verification.
